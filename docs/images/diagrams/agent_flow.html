FridgeAI Agent 对话流程

1
MSG
用户通过 WebSocket 发送消息
"冰箱里有什么? 帮我推荐几道菜" → 前端 WS 客户端发出
ws/chat
2
INV
Agent 调用 get_fridge_inventory 获取库存
返回当前冰箱食材清单 — 鸡蛋×6, 西红柿×3, 牛奶×1, 青椒×2 ...
Tool Call
3
ROU
主 Agent 分析意图并路由到子 Agent
识别为菜谱推荐场景 → 分发至 recipe_expert，传入库存数据与用户偏好
主 Agent → recipe_expert
4
RAG
GraphRAG 混合检索 并行
Neo4j 知识图谱推理 + Milvus 向量语义检索 + BM25 关键词匹配 → 融合排序
Neo4j   Milvus   BM25
5
FLT
偏好过滤与结果融合
去重融合多路结果 → 按用户忌口过滤 → 结合口味偏好排序 → 生成 Top-5
结构化输出
6
LLM
大模型生成个性化推荐文本
DeepSeek V4 Flash 结合检索上下文 + 用户偏好 → 生成贴心推荐回复
DeepSeek V4
7
OUT
流式推送至前端，渲染富文本回复
Token 级 stream_token 推送 → 打字机效果，含菜谱图片、表格、格式文本
stream_token
