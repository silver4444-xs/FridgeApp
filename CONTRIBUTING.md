# Contributing to FridgeAI / 参与贡献

Thanks for your interest in contributing!

## Getting Started / 开始

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/FridgeApp.git`
3. Set up the development environment (see [README Quick Start](README.md#-quick-start))

## Development Workflow / 开发流程

1. Create a feature branch: `git checkout -b feat/your-feature`
2. Make your changes
3. Run tests: `cd Backend && pytest tests/ -v`
4. Commit: `git commit -m "feat: description"`
5. Push: `git push origin feat/your-feature`
6. Open a Pull Request

## Commit Convention / 提交规范

- `feat:` — New feature
- `fix:` — Bug fix
- `docs:` — Documentation
- `refactor:` — Code refactoring
- `test:` — Testing
- `chore:` — Maintenance

## Code Style / 代码风格

- Python: PEP 8, type hints where possible
- Vue: Tab indentation, Options API (uni-app convention)
